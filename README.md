# Сравнение промптов через LLM API

Проект демонстрирует, как один и тот же запрос к LLM меняется при разных способах управления рассуждением.

Платформа для реального API-запуска: GigaChat API от Сбера.

## Что делает скрипт

Берется одна логическая задача:

```text
Улитка находится на дне колодца глубиной 10 метров. Днем она поднимается на 3 метра, ночью сползает на 2 метра. За сколько дней улитка выберется из колодца?
```

Правильный ответ: `8 дней`.

Скрипт решает ее четырьмя способами:

- прямой ответ без дополнительных инструкций;
- ответ с инструкцией `решай пошагово`;
- модель сначала составляет промпт для решения задачи, затем этот промпт используется для решения;
- в промпте создается группа экспертов: аналитик, инженер, критик.

В конце выводится сравнение подходов и вывод о наиболее точном способе.

## Демо без API-ключа

Если нет ключа GigaChat или сертификата Минцифры, можно показать работу задания в mock-режиме:

```bash
npm run ask -- --mock
```

Этот режим не отправляет сетевые запросы, но показывает тот же формат результата: четыре решения одной задачи и сравнение.

## Реальный запуск через API

1. Получите ключ авторизации в GigaChat API:

   https://developers.sber.ru/portal/products/gigachat-api

2. Запустите:

```bash
export GIGACHAT_AUTH_KEY="ваш_ключ_авторизации"
npm run ask
```

Можно передать свою задачу:

```bash
npm run ask -- "У Маши было 5 яблок, она отдала 2 и купила 4. Сколько стало?"
```

## Сертификат для Node.js

Если появляется ошибка `SELF_SIGNED_CERT_IN_CHAIN`, Node.js не доверяет корневому сертификату НУЦ Минцифры, который нужен для GigaChat API.

Запуск с сертификатом:

```bash
export GIGACHAT_AUTH_KEY="ваш_ключ_авторизации"
NODE_EXTRA_CA_CERTS="$HOME/Downloads/russian_trusted_root_ca_pem.crt" npm run ask
```

Временный небезопасный вариант только для локальной проверки:

```bash
NODE_TLS_REJECT_UNAUTHORIZED=0 npm run ask
```

Для реального проекта лучше использовать `NODE_EXTRA_CA_CERTS`, а не отключать TLS-проверку.

## Настройки

По умолчанию используется модель `GigaChat-2`. Можно выбрать другую:

```bash
GIGACHAT_MODEL="GigaChat-2-Pro" npm run ask
```
