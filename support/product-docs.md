# Документация продукта TaskFlow Code Assistant

## Назначение продукта

TaskFlow Code Assistant — это код-ассистент для команд разработки. Он помогает работать с проектной документацией, README, PR, Git-контекстом, RAG-поиском по коду и автоматическим ревью изменений.

Продукт относится к разработке ПО и командной работе с кодом. Поддержка обрабатывает вопросы про:

- авторизацию и доступ к workspace;
- RAG-поиск по документации и коду;
- MCP-инструменты для Git, файлов и backlog;
- автоматическое AI-ревью PR;
- экспорт проектных данных;
- уведомления, тарифы и лимиты продукта;
- интерфейс код-ассистента.

Продукт не продает физические товары и услуги вне разработки: воздушные шары, еду, одежду, бытовые товары, билеты, доставку и похожие внешние запросы. Такие обращения не должны попадать в backlog продукта.

## Роли

В продукте есть роли:

- owner — управляет workspace, оплатой, SSO и пользователями;
- admin — управляет пользователями, проектами и настройками workspace;
- project manager — управляет проектами, задачами и экспортом;
- member — работает с задачами и комментариями;
- viewer — только просматривает задачи.

## Workspace и SSO

SSO включается на уровне workspace. После включения SSO пользователи корпоративного домена должны входить через identity provider. Если домен пользователя не входит в список разрешенных, вход будет отклонен.

## Лимиты

На тарифе Team:

- до 50 пользователей;
- до 25 проектов;
- до 3 одновременных экспортов;
- история задач хранится 180 дней.

На тарифе Business:

- до 500 пользователей;
- до 300 проектов;
- до 20 одновременных экспортов;
- история задач хранится 3 года;
- доступен SSO.

## SLA поддержки

Приоритеты:

- P1 — сервис недоступен для всего workspace, ответ до 15 минут;
- P2 — критичная функция недоступна для части пользователей, ответ до 1 часа;
- P3 — локальная проблема пользователя, ответ до 8 рабочих часов;
- P4 — вопрос или консультация, ответ до 2 рабочих дней.
